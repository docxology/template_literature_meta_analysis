# Syntax Guide

Syntax conventions for manuscript and project documentation in `template_literature_meta_analysis`.

## Links

Use descriptive Markdown links. Prefer repository-root paths in commands and prose, for example `projects/templates/template_literature_meta_analysis/src/literature/models.py`.

## Citations And Cross-References

Use Pandoc citation/crossref syntax:

```markdown
Prior work is cited with [@smith2020]. See [@fig:growth_curve] and [@tbl:hypothesis_scores].
```

Do not hardcode figure, table, or section numbers.

## Manuscript Tokens

Use `{{TOKEN_NAME}}` for values computed from config and generated artifacts. The authoritative token list is the return value of `src/manuscript/variables/compute.py::compute_variables()` plus the manuscript-token tests.

Common token families:

| Family | Source |
| --- | --- |
| `SEARCH_TERM*`, `ENGINE_LIST`, `N_ENGINES` | `manuscript/config.yaml` |
| `CORPUS_SIZE*`, year/growth fields | `output/data/corpus.jsonl`, `temporal_analysis.json` |
| `A1_COUNT` / configured subfield counts | `subfield_classification.json` |
| `HYPOTHESIS_*`, `H*_SCORE` | config + `hypothesis_scores.json` |
| figure counts | `output/figures/` and figure registry |

Detect unresolved tokens before rendering:

```bash
rg "\{\{" projects/templates/template_literature_meta_analysis/output/manuscript
```

## Figures

Figures are generated by `src/visualization/figure_runner.py`. Add figures by:

1. Creating or updating a plotting function in `src/visualization/`.
2. Registering it through the figure runner.
3. Adding tests in `tests/visualization/`.
4. Referencing it from manuscript prose with a Pandoc figure anchor, for example `{#fig:growth_curve}`.
5. Updating `docs/output_inventory.md` when the artifact is part of the contract.

## Tables

Use Pandoc table captions below the table:

```markdown
| Hypothesis | Score |
| --- | ---: |
| H1 | +0.42 |

: Citation-weighted hypothesis scores. {#tbl:hypothesis_scores}
```

## Code Blocks

Always include a language tag:

```bash
uv run pytest projects/templates/template_literature_meta_analysis/tests/ -q
```
