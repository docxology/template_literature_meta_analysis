# Output Directory Conventions

`projects/templates/template_literature_meta_analysis/output/` is generated by
the literature-review pipeline and tracked as a public release-artifact tree
when files stay below the 50 MB public output ceiling. Do not hand-edit output
files; fix the producer and rerun the stage.

## Producer Sequence

```bash
# optional live retrieval; uses configured engines and may need network/keys
uv run python projects/templates/template_literature_meta_analysis/scripts/01_literature_search.py

# deterministic offline analysis from corpus/fixture data
uv run python projects/templates/template_literature_meta_analysis/scripts/02_meta_analysis_pipeline.py

# optional LLM/nanopublication layer; --max-papers 0 reloads existing assertions only
uv run python projects/templates/template_literature_meta_analysis/scripts/03_build_knowledge_graph.py --max-papers 0

# figures
uv run python projects/templates/template_literature_meta_analysis/scripts/04_generate_figures.py --dpi 300

# token-hydrated manuscript copies
uv run python projects/templates/template_literature_meta_analysis/scripts/05_inject_variables.py

# full-text availability report
uv run python projects/templates/template_literature_meta_analysis/scripts/06_fulltext_assessment.py
```

## Layout

| Path | Meaning |
| --- | --- |
| `output/data/corpus.jsonl` | Deduplicated `Paper` records |
| `output/data/*analysis*.json`, `topics.json`, `tfidf_data.json` | Computed analysis inputs for manuscript and figures |
| `output/data/nanopublications.*`, `hypothesis_scores.json` | Optional knowledge-graph outputs |
| `output/figures/*.png` | Headless matplotlib figures |
| `output/figures/figure_registry.json` | Figure registry emitted by the figure runner |
| `output/manuscript/*.md` | Token-resolved manuscript sections used by renderers |
| `output/fulltext/` | Optional downloaded/assessed full-text artifacts |

## Version-Control Policy

- Commit source, config, fixtures, tests, and docs.
- Commit public template `output/` artifacts that stay under the size ceiling.
- Do not commit `.pytest_cache/`, `.venv/`, `.codegraph/`, `__pycache__/`, or egg-info.
- If a generated file is wrong, fix the producer and rerun the stage.
