# Visualization Module

Publication-ready figure generation for the literature meta-analysis.
All figures are saved as PNG at 300 DPI using the non-interactive `Agg` backend.
Every function follows the same contract: accept pre-computed data + output path, return the saved path.

## Modules

### `style.py`
Centralized visual configuration. Defines `VIZ_CONFIG` (figure sizes, DPI, font sizes) and
`SUBFIELD_NAMES` / `HYPOTHESIS_NAMES` label maps. `apply_visual_style()` enforces 16pt minimum
font size and the Wong (2011) colorblind-safe 8-color palette across all figures.

Key exports:
- `VIZ_CONFIG: dict` — global config
- `SUBFIELD_NAMES: dict[str, str]` — subfield key → readable label
- `HYPOTHESIS_NAMES: dict[str, str]` — hypothesis key → readable label
- `apply_visual_style() -> None`

### `field_overview.py`
- `plot_field_summary(total, subfield_counts, output_path)` — horizontal bar chart with counts and percentages
- `plot_subfield_distribution(subfield_counts, output_path)` — donut pie chart (slices < 2% grouped as "Other")

### `temporal_plots.py`
- `plot_growth_curve(year_counts, cumulative, output_path, smoothed_annual)` — dual-axis bar+line; CAGR annotation box
- `plot_subfield_timeline(subfield_year_counts, output_path)` — stacked area chart per domain

### `citation_plots.py`
- `plot_citation_network(graph, output_path, max_nodes)` — spring-layout directed graph; node size ∝ in-degree; community colors
- `plot_degree_distribution(graph, output_path)` — in-degree histogram (log-log for heavy tails)

### `hypothesis_charts.py`
- `plot_hypothesis_dashboard(scores, output_path)` — horizontal diverging bar chart, sorted by consensus strength
- `plot_evidence_timeline(yearly_scores, output_path)` — temporal evidence accumulation lines with ±0.1 neutral band
- `plot_assertion_type_breakdown(assertion_counts, output_path)` — stacked horizontal bars (supports/neutral/contradicts)
- `plot_assertion_summary(total, type_counts, hypothesis_counts, output_path)` — pie + bar multi-panel

### `advanced_plots.py`
- `plot_word_cloud(word_weights, output_path, max_words)` — viridis colormap word cloud
- `plot_pca_embeddings(tfidf_matrix, labels, feature_names, output_path)` — 2-D PCA scatter with loading arrows
- `plot_term_heatmap(tfidf_matrix, feature_names, labels, output_path, n_terms)` — YlOrRd heatmap; terms selected by **between-subfield variance** (discriminative, not globally frequent)
- `plot_dendrogram(tfidf_matrix, labels, output_path)` — Ward-linkage hierarchical clustering of subfield centroids
- `plot_topic_term_bars(topics, output_path)` — faceted NMF topic-term bar charts
- `plot_cooccurrence_matrix(documents, output_path, n_terms)` — symmetric normalized co-occurrence heatmap

## Term Selection in `plot_term_heatmap`

Terms are ranked by **between-subfield variance** of mean TF-IDF weights, not by global mean.
This selects discriminative terms that differentiate domains (e.g. "cortex" for C1 neuroscience,
"robot" for C2 robotics) rather than ubiquitous field terms ("inference", "model") that appear
uniformly across all subfields.
